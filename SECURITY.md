# Security Policy

## Supported Use

This project is actively maintained as a public portfolio and open-source repository. Security fixes are handled on a best-effort basis.

## Reporting a Vulnerability

Please do not report security issues in public GitHub issues.

Instead, contact the maintainer directly and include:

- a clear description of the issue
- reproduction steps
- impact assessment
- any suggested fix if available

If the issue involves accidental key exposure, rotate the affected key immediately.
